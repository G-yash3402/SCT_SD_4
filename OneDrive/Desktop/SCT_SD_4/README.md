# SCT_SD_4 - Product Data Scraper

A Python web scraping project that extracts product information such as product names, prices, and ratings from an e-commerce website and stores the data in a CSV file.

## Features
- Extracts product names
- Extracts prices
- Extracts ratings
- Saves data to CSV file

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Output
Data is stored in products.csv.
